# LoRA学習プロンプト工房

LoRAに覚えさせたい特徴と、固定させたくない属性を分け、学習画像セット全体の配分を設計する静的Webアプリです。

## 主な機能

- 最大剰余方式による均等・指定比率配分
- 最低／最大件数、禁止ペア、最大連続回数、乱数シード
- 環境別の数値重み・括弧強調・ネガティブ出力
- プロンプト本文だけをコピーする独立カード
- 生成済み／採用／保留／不採用と不採用理由の管理
- 採用画像だけでの再集計と、不足属性を優先した追加生成
- 偏り診断、LocalStorage自動保存、JSONバックアップ／復元

画像や入力内容を外部サービスへ送信しません。画像ファイル自体の管理やAI画像解析はMVP対象外です。

## ローカル確認

Node.js 22.13以上を用意し、依存関係をインストールしてから開発サーバーを起動します。

```bash
pnpm install
pnpm dev
```

自動テストと本番ビルドは次で確認できます。

```bash
pnpm test
pnpm build
```

## GitHub Pages

このリポジトリをGitHubへpushし、Repository Settings → Pages → Source で **GitHub Actions** を選びます。`main`ブランチへのpush時に同梱のワークフローがビルド・公開します。
